# ImageBankManager

[![Python](https://img.shields.io/badge/python-3.11.x-blue.svg)](https://www.python.org/)
[![Django](https://img.shields.io/badge/django-latest-092E20.svg)](https://www.djangoproject.com/)

ImageBankManager is a project based on [Finder v2](https://github.com/diogotoporcov/Finder2), reimplementing it using Django for the backend and planning to use React with TypeScript for the frontend.
